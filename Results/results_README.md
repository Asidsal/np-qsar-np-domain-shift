# Results

This directory receives all outputs generated by `notebooks/02_modeling_pipeline.ipynb`.
It is empty in the repository. Run Notebook 02 to populate it.

All generated figures and tables are also available in the Zenodo deposit
(DOI: to be assigned upon acceptance).

---

## Expected Output Files

### Results Tables

| File | Description | Key columns |
|------|-------------|-------------|
| `Table1_full_results.csv` | All 36 model evaluations (4 models × 3 splits × 3 targets) | Target, Split, Model, n_train, n_test, MCC, AUROC, AUPRC, BalAcc |
| `Table2_mcc_summary_pivot.csv` | MCC pivot with degradation scores | Target, Model, random, scaffold, np_holdout, degradation |
| `Table3_correlation_results.csv` | Spearman correlations (Tanimoto vs performance) | Target, Spearman_r, Spearman_p, N_NP_test |
| `multiseed_results.csv` | Multi-seed stability (5 seeds, RF only) | Target, MCC_mean, MCC_std, AUROC_mean, AUROC_std |
| `sensitivity_coconut_only.csv` | Subgroup performance: COCONUT-confirmed vs NP-score-only | Target, Group, N, Active_pct, MCC, AUROC, Confidence |
| `rebalance_results.csv` | Activity-rebalanced COCONUT evaluation | Target, N_coconut, Active_pct_orig, MCC_original, MCC_rebalanced, MCC_std |
| `tanimoto_by_criterion.csv` | Tanimoto similarity by NP classification criterion | Target, N_coconut, Median_tan_coconut, N_npscore, Median_tan_npscore, p_coconut_vs_npscore |
| `reproducibility_report.txt` | Full environment log | Python version, package versions, configuration, dataset sizes |

### Checkpoint Files (subdirectory)

Notebook 02 saves intermediate checkpoints to `results/checkpoints/` to protect
against Colab session resets:

| File | Contents |
|------|----------|
| `{target}_descriptors.pkl` | Morgan, MACCS, RDKit2D descriptor matrices per target |
| `{target}_tanimoto.npy` | Max Tanimoto similarity array (NP test vs synthetic train) |

These files are not included in the Zenodo deposit. They can be regenerated by
re-running Cells 03 and 04 of Notebook 02.
