# Data

This directory contains curation metadata and summary statistics.
The full curated CSV files are not included in this repository due to size
and licensing constraints. They can be reproduced by running
`notebooks/01_data_curation.ipynb` against the source databases, or
downloaded from this repo.

---

## Source Data

### ChEMBL 36

- **URL:** https://ftp.ebi.ac.uk/pub/databases/chembl/ChEMBLdb/releases/chembl_36/
- **File needed:** `chembl_36_sqlite.tar.gz` (~4 GB compressed, ~23 GB extracted)
- **License:** CC BY-SA 3.0
- **Citation:** Gaulton A, et al. ChEMBL: a large-scale bioactivity database for drug discovery. Nucleic Acids Res. 2012;40(D1):D1100-7.

### COCONUT 2.0

- **URL:** https://coconut.naturalproducts.net/download
- **File needed:** `coconut_csv_lite_05-2026.zip` (~191 MB)
- **Select:** Natural Products - CSV format (lite version)
- **Version:** May 2026 release
- **License:** CC0 (no restrictions)
- **Citation:** Chandrasekhar V, et al. COCONUT 2.0: a comprehensive overhaul and curation of the collection of open natural products database. Nucleic Acids Res. 2024;gkae1063.

## Curation Summary

After standardization and cross-set deduplication:

| Target | Synthetic train | COCONUT-confirmed NP | NP-score-only | Total |
|--------|----------------|---------------------|---------------|-------|
| AChE | 4,248 | 214 | 198 | 4,660 |
| 5-LOX | 1,724 | 119 | 282 | 2,125 |
| COX-2 | 3,513 | 115 | 56 | 3,684 |
| **Total** | **9,485** | **448** | **536** | **10,469** |

Activity threshold: pChEMBL ≥ 6.0 (IC50/Ki ≤ 1 μM) = active; pChEMBL < 6.0 = inactive.

NP classification: a compound is COCONUT-confirmed if its InChIKey appears in COCONUT 2.0.
A compound is NP-score-only if its Ertl NP-likeness score ≥ 0.5 but it does not appear in COCONUT.
Compounds satisfying both criteria are assigned to the COCONUT-confirmed group.
The two evaluated groups are mutually exclusive.

## Files in This Directory

### `curation_log.txt`

Generated by Notebook 01 (Cell 08). Documents all curation decisions:
- pChEMBL thresholds
- Standardization approach (RDKit: largest fragment, neutralization, canonical SMILES)
- Duplicate handling (median pChEMBL per InChIKey)
- NP classification criteria
- Cross-set deduplication method (full InChIKey + connectivity layer)
- Final dataset sizes per target

### `dataset_summary.csv`

Compound counts per target and compound group. Useful for verifying
successful reproduction of the curation pipeline.

Columns: Target, Synthetic_n, Synthetic_active, NP_n, NP_active, Combined_n

### `deduplication_report.csv`

Documents cross-set deduplication results per target.

Columns: Target, NP_before, Exact_removed, Tautomer_removed, NP_after

## Reproducing the Curated Datasets

1. Download ChEMBL 36 SQLite and extract: `tar -xzf chembl_36_sqlite.tar.gz`
2. Download COCONUT 2.0 lite CSV and extract
3. Open `notebooks/01_data_curation.ipynb`
4. Edit `CONFIG` in Cell 01 with your local file paths
5. Run all cells in order (estimated 45-60 minutes)
6. Output CSVs will be written to your configured `output_dir`

Expected output files per target (9 files total):
- `{target}_synthetic_train.csv`
- `{target}_np_test.csv`
- `{target}_combined.csv`

Where target is AChE, 5LOX, or COX2.
